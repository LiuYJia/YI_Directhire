#易·直聘
+ [Github](https://github.com/LiuYJia/YI_Directhire)
## 项目简介
**易·直聘**是一款面向职场人员的APP，在互联网迅速发展的大背景下，新一代的人逐渐成为劳动市场的主力军，随着获取信息方式的改变，在招聘与求职方面出现了以往不同的方式，这一代人更愿意从相关APP软件商获取相关信息，**易·直聘**便旨在提供这一信息来源。
